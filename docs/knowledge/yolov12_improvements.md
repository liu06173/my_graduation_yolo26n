# YOLOv12 革新无人机极小目标检测 — 知识库

> 来源: 计算机视觉研究院 微信公众号 (2025)  
> 论文: https://pmc.ncbi.nlm.nih.gov/articles/PMC12808633/pdf/41598_2025_Article_31803.pdf

## 一、核心痛点

无人机航拍目标检测的核心瓶颈：**目标仅占3像素以内**时，传统YOLO模型直接失效：
- 纹理、边缘特征几乎消失
- 深度卷积下采样易丢失细粒度空间信息
- 轻量化与高精度难以兼顾
- 多尺度特征融合不足，小目标极易被背景淹没

---

## 二、四大创新模块

### 2.1 C3K2 轻量骨干模块
- 替换传统 3×3 卷积为 **2×2 卷积**
- 大幅降低参数量与计算量
- 同时保留细粒度空间特征，解决卷积冗余问题

### 2.2 A2C2F 区域注意力模块
- 融合 **多头MLP + 局部区域注意力**
- 聚焦极小目标的**局部空间依赖**
- 摒弃全局注意力的高算力消耗
- 强化弱目标特征、抑制背景噪声

### 2.3 多尺度特征融合策略
- 堆叠 A2C2F 模块 + Concat + Upsample
- 保留高分辨率特征
- 小目标不丢失、不模糊

### 2.4 解耦检测头 (Decoupled Head)
- 分类、回归分支分离
- 搭配注意力引导特征融合
- 提升预测一致性与推理速度

## 三、实验结果 (VisDrone)

| 指标 | 数值 |
|------|------|
| Precision | **69.1%** |
| Recall | **48.5%** |
| F1 Score | **56.99%** |
| mAP@50 | **58.8%** |
| mAP@50-95 | **40.9%** |
| FPS (A100) | **40 FPS** |
| 参数量 | 59.1M |
| GFLOPs | 198.6 |

**关键结论**: 对小于3像素的超小目标检测能力，远超YOLOv8/v9及各类Transformer检测模型。

---

## 四、对我们的启示

对比我们当前的实验：

| 指标 | 我们的YOLOv11n-640 | YOLOv12改进版 (VisDrone全10类) |
|------|-------------------|-------------------------------|
| mAP@50 | 0.548 (2类) | 0.588 (10类) |
| 参数量 | 2.59M | 59.1M |
| 输入尺寸 | 640 | ? |

**可借鉴的改进方向:**
1. **A2C2F注意力**: 替代我们实验中负优化的ECA。A2C2F是局部区域注意力，专门针对小目标，比全局通道注意力更有效
2. **2×2卷积**: C3K2用2×2替代3×3，减少参数量同时保留空间细节
3. **解耦检测头**: 分类和回归分离，可能改善我们P-R不均衡的问题
4. **多尺度融合**: 堆叠注意力+Concat，保留高分辨率特征

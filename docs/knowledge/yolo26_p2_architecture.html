🏗️ 骨干网络 (Backbone)

输入 512×512×3 │ ├─ [0] Conv(64, s=2) ─── 256×256 # P1/2 ├─ [1] Conv(128, s=2) ─── 128×128 # P2/4 ├─ [2] C3k2_ECA(256) ← ECA通道注意力 ├─ [3] CoordAtt ← 坐标位置注意力 ├─ [4] Conv(256, s=2) ─── 64×64 # P3/8 ├─ [5] C3k2_ECA(512) ← ECA通道注意力 ├─ [6] CoordAtt ← 坐标位置注意力 ├─ [7] Conv(512, s=2) ─── 32×32 # P4/16 ├─ [8] C3k2(512) ├─ [9] Conv(1024, s=2) ─── 16×16 # P5/32 ├─ [10] C3k2(1024) ├─ [11] SPPF(1024) # 多尺度池化 └─ [12] C2PSA(1024) # 位置自注意力
设计要点：ECA + CoordAtt 仅部署在 P2/4 和 P3/8 两层。 高分辨率层包含最多小目标信息，注意力收益最大； P4/P5 由 C2PSA 提供位置自注意力，无需重复叠加。

⚡ 六大改进模块详解

通道注意力 ECA
conv.py:676 · ECCV 2020
核心思想：用1D卷积替代全连接层做通道交互，参数量接近于零。
自适应kernel： k = |log₂(C)/2 + 0.5|，取奇数
流程： AdaptiveAvgPool2d → Conv1d(k) → Sigmoid → ×原特征
额外参数：仅 k 个float (通常3~5个)
位置注意力 CoordAtt
conv.py:707 · CVPR 2021
核心思想：将2D全局池化分解为1D水平和垂直编码，保留精确空间坐标。
流程： X方向池化 + Y方向池化 → 拼接 → Conv降维 → 分离 → 双Sigmoid门控
优势：不抹去位置信息，对小目标定位极为关键
动态上采样 DySample
conv.py:771 · ICCV 2023
核心思想：从特征内容学习采样偏移量，动态调整上采样位置。
流程： Pool(8×8) → Conv→SiLU→Conv → 生成offset → grid_sample
vs最近邻：保留更多细节，小目标边缘更清晰
加权融合 WeightedConcat
conv.py:745 · CVPR 2020 (BiFPN)
核心思想：不同层特征学习不同融合权重，而非等权拼接。
实现： w = softmax(learnable_params)，然后加权拼接
例子： P3=0.7×feat_P3 + P4=0.3×feat_P4 (自动学习)
ECA残差块 ECABottleneck
block.py:486
结构： Conv(1×1) → Conv(3×3) → ECA → +残差连接
与标准Bottleneck区别：第二个Conv后插入ECA通道注意力
额外成本：每个Block仅增~5个参数
ECA增强CSP C3k2_ECA
block.py:1140
结构：继承C2f，Bottleneck替换为ECABottleneck
逻辑： attn=False, c3k=False → 走ECA路径
部署位置： Backbone P2和P3层，替代标准C3k2
